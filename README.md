# XBRuleView

直尺——自定义刻度、刻度颜色、垂直水平方向等.

支持Objective-C和swift

## 集成方式

### Objective-C

  直接将XBRuleView和XBRuleContentView的.h和.m文件拖进工程即可。具体参照demo工程
  
### Swift

  直接将XBSwiftRuleView.swift文件拖进工程即可。具体参照SwiftTestView.swift文件和demo工程
